# How to use ?              
--------------------------
> Exemple : ./wolf3d map

### Description

> You can choose your maps in the "maps" folder :

List :
- maps/mindfuck.map
- maps/big_cube.map
- maps/lnieto_1.map
- maps/prison_escape.map
- maps/test_texture.map
- maps/test_tp.map

> or create your own map.
